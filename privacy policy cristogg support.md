# Informativa sulla Privacy per il Bot Discord Cristo.gg Support.

La presente Informativa spiega come vengono raccolti, memorizzati e trattati i dati degli utenti che interagiscono con il bot Cristo.gg Support (di seguito "il Bot").

## 1. Dati Raccolti
Per funzionare correttamente, il Bot può accedere e memorizzare temporaneamente o permanentemente i seguenti dati forniti dall'API di Discord:
* ID Utente, ID del Server (Guild ID) e ID dei canali.
* Nome utente e avatar.
* Dati di configurazione inseriti tramite comandi (es. canali di log, ruoli).
* Messaggi di testo inviati nei comandi o log di sistema.

## 2. Finalità del Trattamento
I dati vengono raccolti esclusivamente per permettere il corretto funzionamento delle feature del bot (es. sistemi di moderazione, economia, utility) e non verranno mai venduti, condivisi o ceduti a terze parti.

## 3. Conservazione dei Dati
I dati di configurazione rimangono memorizzati nel database finché il Bot fa parte del server o finché l'utente non ne richiede l'eliminazione. I dati temporanei (come la cache) vengono eliminati regolarmente.

## 4. Diritti dell'Utente (Rimozione Dati)
Gli utenti possono richiedere la rimozione immediata dei propri dati memorizzati nel database in qualsiasi momento. Per farlo, è sufficiente [scrivere sul server di supporto / inviare un messaggio a `@_wizzo__`].

## 5. Aggiornamenti
Questa informativa può cambiare nel tempo per riflettere gli aggiornamenti del codice del bot. L'uso continuato del Bot costituisce l'accettazione della presente policy.
